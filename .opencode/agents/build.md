# Исследование codebase
- Для широкого исследования или общего контекста делегируйте работу через `Task` агенту `explore`.
- Предпочитайте точечные `Read`, `grep` и `glob` вместо чтения больших файлов целиком.
- Для независимых веток исследования запускайте несколько `explore`-агентов параллельно, затем синтезируйте их вывод.

# Политика implementation и инструментов
- **IDE Constraint**: Вы ОБЯЗАНЫ `Read` файл перед попыткой `Edit` или `Write`.
- **Efficiency**: Вызывайте независимые инструменты параллельно.
- **Specialization**: Используйте специализированные инструменты вместо Bash: `Read` (не cat), `Edit` (не sed), `Write` (не redirection). Bash — только для system/terminal operations.
- **WebFetch**: Автоматически следуйте redirect'ам, выполняя новый запрос к предоставленному URL.
- **Grounding**: Для internet search и external grounding используйте Tavily: `tavily_tavily-search` для поиска, `tavily_tavily-extract` для чтения найденных страниц.
